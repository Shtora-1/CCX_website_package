CCX Web Package
===============

Archivos incluidos:
- index.html
- style.css
- script.js
- assets/logo.png
- CCX_Presentation.pdf

Cómo usar:
1) Descomprime el ZIP.
2) Abre index.html en tu navegador.
3) Reemplaza assets/logo.png por tu logo real, y añade CCX.jpg si quieres usar imagen de hero.

Notas:
- Tema seleccionado: Apple-like minimal (claro con modo oscuro).
- Si quieres que genere un ZIP con imagenes optimizadas o un PDF del sitio renderizado, dime y lo intento.
